Lost connection to "iPhone4s"

发表于 2016-10-14 | 分类于 日常Bug | | 阅读次数

坑爹:

前几天不知什么原因，项目突然开始频繁的报
Lost connection to “iphone 4s”
restore the connection to “iphone4s” and run “XX” agian,or if “XX”is still running,you can attach to it by
selecting Debug >Attach to Progress >XX.
起初我以为是线虚，，后来一查居然是内存暴增，检查了两天也没个结果，第二天晚上拿别人的手机试了一下居然没问题，看来只能是手机的问题。

没有用:

平时一直是在桌面直接删除应用的，重新测试还是偶尔闪退，后来在设置>通用>存储空间与iCloud用量>管理存储空间 中删除应用，事实证明这个办法不行，又突然就闪了。到底什么鬼
浪费了两天时间，原因至今不明，纪录一哈，待续。。。

求解决方案:

应该只可能是内存暴增了，那天晚上用别人的是6，可能性能要好一点，，但是拿以前公司第一版的也是频繁报这个，以前是根本不会弹这个框的，费解ing’’’

11月10日更新:

最近忙着重构，写的也差不多了，想起这个bug，又特地来试了一下，居然好了？excuse me！！原因至今不明，或许是我把手机的名字从“板砖4s”改成“高大上的4s”的缘故吧～～
