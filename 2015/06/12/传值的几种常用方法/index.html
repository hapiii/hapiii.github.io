传值的几种常用方法

发表于 2015-06-12 | | 阅读次数

方法1.代理传值
1>代理方法：
@protocol ChangeDelegate
//代理的方法以及要传的值

  • (void)changeBut:(UIButton )but Color:(UIColor )color;

@end
2>代理指针
@property (nonatomic,assign) id delegate;
3>调用代理的方法；
[_delegate changeBut:but Color:but.backgroundColor];
4>在要传值的页面添加代理


5>设置代理
vc.delegate = self;
6>实现代理的方法

#pragma mark - 代理的实现方法；

  • (void)changeBut:(UIButton )but Color:(UIColor )color{
    [_Cbut setBackgroundColor:color];

}
方法2.block方法，也是我最喜欢的一种方法
1>声明block
typedef void(^ChangeColor)(UIColor color);
2>设置属性
@property (nonatomic,copy)ChangeColor block;
3>block传值
_block(but.backgroundColor);
4>在要传达的页面调用block
vc.block = ^(UIColor color){
but.backgroundColor = color;
};

方法3.通知 常用来一对多，虽然简单，但是不太经常要用到
1>发送消息
[[NSNotificationCenter defaultCenter]postNotificationName:@”33” object:but.backgroundColor];

2>在要传达的页面添加观察者并执行方法
[[NSNotificationCenter defaultCenter]addObserver:self selector:@selector(changebutColor:) name:@”33” object:nil];

3>收到消息通知要执行的方法

  • (void)changebutColor:(NSNotification *)color{
    [_Cbut setBackgroundColor:color.object];

}

方法4.单例 单例也可以用来传值 不过太大材小用，
1>创造一个单例ButManger并添加属性和方法
@property(nonatomic,strong)UIColor *color;

  • (instancetype)shareColor;
    在.m里实例化
  • (instancetype)shareColor{

static ButManger *manger = nil;
if (!manger) {
manger = [[ButManger alloc]init];
}
return manger;
}
3>给单例属性赋值

ButManger *manger = [ButManger shareColor];
manger.color = but.backgroundColor;
4> 单例的实现；

  • (void)viewWillAppear:(BOOL)animated{
    [super viewWillAppear:animated];
    //单例的传值;
    ButManger *manger = [ButManger shareColor];
    if (manger.color) {
    // _Cbut.backgroundColor = manger.color;
    }

}
